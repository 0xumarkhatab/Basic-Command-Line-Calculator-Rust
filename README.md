# Basic-Command-Line-Calculator-Rust
This Repository is dedicated to make the basic calculator purely by Rust

## Project Execution Instructions

### Pre-Requisites
  - Rust Up Compiler

#### Run Prokect 
   Run 
   <code> cargo new calculator</code>
   
   After Successful execution of the above command, run 
   
   <code> cd calculator</code>
   
   in the src  directory , edit the main.rs file and copy the code into the file.
   
   Now Run
   
   <code>cargo run -- num1 operator num2 </code>
  
   Replace the 'num1' and 'num2' with any Float.
   
   Check the Result and star my Repository :)
